# Hotel Booking System

## 📌 Description
A web-based hotel booking system that allows users to browse available rooms, search for accommodations, and make reservations. The system also provides basic management features for handling bookings and customer information.

---

## 🚀 Features
- View available hotel rooms
- Search rooms by criteria
- Book rooms online
- Manage booking information
- Contact form for customer inquiries
- Basic admin management (if applicable)

---

## 🛠 Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP (Apache Server)

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/PhuongUyen-118/hotel-booking-system.git
2. Move the project to XAMPP folder:
  Copy project to: xampp/htdocs/
3. Start XAMPP:
  Run Apache
  Run MySQL
4. Import database:
  Open phpMyAdmin
  Create a new database
  Import file from /sql/ folder
5.Open in browser:
  http://localhost/hotel_booking
